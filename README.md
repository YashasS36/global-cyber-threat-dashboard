🛡️ Cyber Threat Intelligence Dashboard
Overview

The Cyber Threat Intelligence Dashboard is an interactive Power BI project designed to analyze cybersecurity incidents, threat severity, attack trends, affected industries, and geographic impact. This dashboard transforms raw cyber threat data into actionable insights, helping users identify patterns, monitor risks, and understand the evolving threat landscape.

Project Objectives
Analyze cyber threats across different countries and regions.
Monitor incident severity levels and threat categories.
Identify industries most affected by cyber attacks.
Track cyber incidents over time.
Discover emerging threat patterns and high-risk areas.
Enable data-driven cybersecurity decision-making.
